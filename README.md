# Employee Attrition MLOps System 🚀

Production-ready end-to-end ML classification system predicting HR employee attrition.

## Architecture
```
Attrition/
├── data/          # Raw & processed data
├── src/           # Core Python modules
├── frontend/      # Streamlit UI
├── tests/         # Unit tests
├── deployment/    # Docker & infra
├── docs/          # Academic report
├── .github/       # CI/CD workflows
└── TODO.md        # Progress tracker
```

## Quick Start
```bash
pip install -r requirements.txt
cp .env.example .env  # Fill credentials
make setup
make train
make api
make frontend
```
